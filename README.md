# Library Management System (Java)

## Description
This is a simple console-based Library Management System developed using Java.
The project allows users to manage books in a library using a command-line interface.

## Features
- Add new books
- View all books
- Search books by ID or name
- Remove books
- Input validation and error handling

## Technologies Used
- Java
- OOP Concepts (Classes, Objects)
- Collections (ArrayList)
- Command Line Interface (CLI)

## How to Run
1. Compile all Java files
2. Run the Main.java file
3. Follow the on-screen menu
